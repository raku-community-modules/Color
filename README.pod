=head1 NAME

Color - Format conversion and math operations on colours

=head1 SYNOPSIS

    use Color;

    my $white        = Color.new(255, 255, 255);
    my $almost_black = Color.new('#111');

    my $gray = $white / 2;
    say $gray.hex; # prints "#808080"

    say $almost_black + 25; # prints "42, 42, 42"

=head1 DESCRIPTION

This module allows you to perform mathematical operations on RGB colour tuples,
as well as convert them into other colour formats, like hex.

=head1 METHODS

=head2 C<new>

    my $rgb = Color.new;

Creates new C<Color> object.
